# 评论系统使用指南

本文档介绍如何在VitePress文档站点中使用内置的Waline评论系统。

## 评论系统简介

本网站已集成[Waline](https://waline.js.org/)评论系统，该系统基于Valine开发，提供了更丰富的功能和更好的用户体验。Waline评论系统支持以下特性：

- 用户可以通过昵称、邮箱和网址进行评论
- 支持Markdown语法
- 支持表情、代码高亮等高级功能
- 支持深色模式自动适配
- 支持评论分页展示

## 如何使用评论系统

### 发表评论

1. 滚动到页面底部，找到评论区域
2. 填写您的昵称、邮箱（选填）
3. 在评论框中输入您的评论内容
4. 点击「提交评论」按钮

### 评论格式

评论支持基本的Markdown语法，例如：

- **粗体文本**：使用 `**粗体文本**`
- *斜体文本*：使用 `*斜体文本*`
- 链接：使用 `[链接文本](链接地址)`
- 代码：使用 `行内代码` 或代码块

### 评论设置

本网站的评论系统已配置以下参数：

- 服务器地址：`https://waline.020417.xyz/`
- 深色模式：自动适配
- 必填信息：昵称、邮箱（可选）
- 评论占位符：「欢迎发表您的评论！」
- 每页评论数：10条

## 在特定页面启用/禁用评论

如果您希望在特定页面启用或禁用评论功能，可以在页面的frontmatter中进行设置：

### 禁用页面评论

在Markdown文件的开头添加以下配置：

```markdown
---
title: 页面标题
description: 页面描述
comment: false
---
```

### 启用页面评论

如果全局配置中关闭了评论，但希望在特定页面启用：

```markdown
---
title: 页面标题
description: 页面描述
comment: true
---
```

## 常见问题

### 评论不显示怎么办？

- 确保您的网络连接正常
- 检查是否有JavaScript错误（可通过浏览器开发者工具查看）
- 刷新页面后重试

### 评论提交失败怎么办？

- 检查必填字段是否已填写
- 确认邮箱格式是否正确（如果填写）
- 检查是否包含敏感内容
- 稍后再试或联系管理员
